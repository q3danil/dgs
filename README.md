# 📊 Data Generator System

Сервис для генерации синтетических данных для тестирования систем. Позволяет создавать датасеты через Web-интерфейс, CLI-клиент или автоматизированные YAML-конфиги.

## 🌟 Основные возможности
- **AI-генерация**: Создание сложных контекстных данных с помощью LLM
- **Типизация данных**: Поддержка строгих типов (int, float, str, bool) для корректного импорта
- **Многослойный интерфейс**:
  - **CLI Client**: Инструмент для работы в терминале.
  - **Web Portal**: Удобный UI для быстрой работы.
  - **RESET API**: Документация Swagger
- **Экспорт**:
  - Прямой экспорт в **CSV**.
  - Интеграция с **PostgreSQL** (локальные или внешние инстансы).

## 🏗 Архитектура системы
Проект разделен на три компонента:
- **Backend (FastAPI)**: Ядро генерации и API шлюз.
- **Frontend (Nginx)**: Интерфейс для конечных пользователей.
- **Client (CLI)**: Автономное Python-приложение.
